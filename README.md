### DIO_ANIMES:japanese_goblin:

- Nesse Sistema  de Cadastro de Animes, adaptado do Sistema de Cadastro de Séries da DIGITAL INOVATION ONE, busquei dar ao usuário a opção de escolher entre os diversos gêneros de animes existentes, dentre os quais se destacam o Shonen( de animações como Dragon Ball e Naruto) e o Seinen(de Berserk e Neon Genesis Evangelion).
